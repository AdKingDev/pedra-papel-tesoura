# pedra-papel-tesoura
Projeto inspirado no clássico jogo Pedra, papel e tesoura. O jogo possui um design moderno e interativo, desenvolvido como parte dos meus estudos em lógica de programação e desenvolvimento web.
O projeto simula partidas entre jogador e computador, com interface intuitiva e feedback visual em tempo real.

Funcionalidades:
- Escolha entre Pedra, Papel ou Tesoura
- Sistema de jogada contra o Computador
- Resultado automático da rodada
- Atualização do placar em tempo real
- Interface simples e intuitiva
- Feedback visual de vitória, derrota ou empate

MAIOR DESAFIO:
O meu maior desafio nesse projeto foi durante a interação do JavaScript, sendo mais preciso durante a criação do botão para tornar o jogo mais acessivel, deixando ele com a opção de alternar entre o tema Claro e o tema Escuro.
